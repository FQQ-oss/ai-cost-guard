# AI Cost Guard - 数据库设计文档

## 一、概览

- **数据库名**：`ai_cost_guard`
- **字符集**：`utf8mb4`
- **排序规则**：`utf8mb4_general_ci`
- **共 16 张表**，按功能分 6 组

| 分组 | 表 | 说明 |
|---|---|---|
| 系统管理（5） | sys_user, sys_role, sys_user_role, sys_team, sys_team_member | 用户、角色、团队 |
| 项目与Key（3） | project, llm_key_pool, project_api_key | 项目、LLM Key池、代理Key |
| 模型与路由（2） | model_pricing, route_rule | 定价、路由规则 |
| 请求与统计（2） | request_log, usage_stats_daily | 请求日志、日用量统计 |
| 缓存与告警（3） | cache_config, alert_rule, alert_history | 语义缓存配置、告警 |
| 审计（1） | audit_log | 操作审计日志 |

## 二、ER 关系

```
sys_user ──1:N── sys_user_role ──N:1── sys_role
sys_user ──1:N── sys_team_member ──N:1── sys_team
sys_team ──1:N── project
project  ──1:N── project_api_key
project  ──1:N── request_log
project  ──1:N── usage_stats_daily
project  ──1:1── cache_config
alert_rule ──1:N── alert_history
```
