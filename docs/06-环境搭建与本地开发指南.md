# AI Cost Guard - 环境搭建与本地开发指南

## 一、环境要求

| 软件 | 版本要求 | 用途 |
|---|---|---|
| JDK | 17+ | 后端运行环境 |
| MySQL | 8.0+ | 数据持久化 |
| Redis | 6.0+ | 缓存/限流/会话 |
| Maven | 3.9+ | 后端依赖管理与构建 |
| Node.js | 18+ | 前端构建 |
| npm | 9+ | 前端包管理 |
| IntelliJ IDEA | 2024+ | 推荐 IDE |
| Git | 2.x | 版本控制 |

## 二、环境安装（Windows）

### 2.1 JDK 17

推荐 Microsoft OpenJDK：
1. 下载地址：https://learn.microsoft.com/zh-cn/java/openjdk/download
2. 安装后配置环境变量：
```
JAVA_HOME = C:\Users\{用户名}\.jdks\ms-17.0.18   （或你的安装路径）
Path 追加 %JAVA_HOME%\bin
```
3. 验证：`java -version`

### 2.2 MySQL 8

1. 下载安装 MySQL 8.0+
2. 创建 root 用户密码（本项目 dev 环境密码为 `root`）
3. 验证：`mysql -u root -p`

### 2.3 Redis

Windows 方案：
- 方案 A：安装 Redis for Windows（Memurai 或 tporadowski/redis）
- 方案 B：WSL2 安装 `sudo apt install redis-server`
- 方案 C：Docker `docker run -d -p 6379:6379 redis`

验证：`redis-cli ping` → 返回 `PONG`

### 2.4 Node.js

下载安装 Node.js 18 LTS：https://nodejs.org
验证：`node -v && npm -v`

## 三、项目拉取与初始化

### 3.1 克隆代码
```bash
git clone <仓库地址>
cd ai-cost-guard
```

### 3.2 初始化数据库
```bash
mysql -u root -p < sql/init.sql
```

该脚本会：
- 创建数据库 `ai_cost_guard`
- 创建 16 张表
- 插入初始数据（3 个角色、1 个管理员、8 条模型定价）

> 注意：init.sql 中的 admin 密码哈希是无效的，需要通过注册接口创建可用账号。

### 3.3 确认 Redis 已启动
```bash
redis-cli ping
# 应返回 PONG
```

### 3.4 配置文件检查

主配置：`src/main/resources/application.yml`
环境配置：`src/main/resources/application-dev.yml`

需确认 `application-dev.yml` 中以下配置与本机一致：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/ai_cost_guard?...
    username: root
    password: root          # ← 改成你的 MySQL 密码
  data:
    redis:
      host: localhost
      port: 6379
      password:             # ← 如果 Redis 有密码则填写
```

## 四、启动项目

### 4.1 后端启动

**方式一：IntelliJ IDEA（推荐）**
1. 用 IDEA 打开项目根目录
2. 等待 Maven 自动导入依赖（右下角进度条）
3. 如果依赖未自动下载：右键 `pom.xml` → Maven → Reload Project
4. 确认 Project SDK 为 JDK 17：File → Project Structure → Project
5. 运行 `AiCostGuardApplication.java` 的 main 方法
6. 看到 `Started AiCostGuardApplication` 即启动成功，端口 8080

**方式二：命令行**
```bash
# 设置 JAVA_HOME（如果未全局配置）
export JAVA_HOME="C:/Users/{用户名}/.jdks/ms-17.0.18"

# 使用 Maven Wrapper
./mvnw spring-boot:run

# 或直接打包运行
./mvnw clean package -DskipTests
java -jar target/ai-cost-guard-1.0.0.jar
```

### 4.2 前端启动
```bash
cd frontend
npm install       # 首次需安装依赖
npm run dev       # 启动开发服务器
```
访问 http://localhost:3000

### 4.3 注册第一个用户
```bash
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"123456","nickname":"Admin"}'
```

然后在前端 http://localhost:3000 用 admin / 123456 登录。

### 4.4 查看 API 文档

Knife4j (Swagger UI) 地址：http://localhost:8080/doc.html

## 五、IDEA 常见问题

### 问题 1：不支持发行版本 5
**解决**：File → Project Structure → Project → Language Level 选 17

### 问题 2：程序包 xxx 不存在
**解决**：右键 pom.xml → Maven → Reload Project

### 问题 3：Lombok 报红
**解决**：
1. Settings → Plugins → 搜索安装 Lombok 插件
2. Settings → Build → Compiler → Annotation Processors → 勾选 Enable

### 问题 4：Maven 依赖下载失败
**解决**：检查网络，或配置 Maven 镜像：
```xml
<!-- ~/.m2/settings.xml -->
<mirrors>
  <mirror>
    <id>aliyun</id>
    <url>https://maven.aliyun.com/repository/public</url>
    <mirrorOf>central</mirrorOf>
  </mirror>
</mirrors>
```
