# AI Cost Guard - 接口文档

## 一、通用说明

### 基础信息
- **Base URL**：`http://localhost:8080/api`
- **认证方式**：Bearer Token（JWT），Header: `Authorization: Bearer {token}`
- **免认证接口**：`/api/auth/login`, `/api/auth/register`, `/api/v1/chat/**`

### 统一响应格式
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

### 错误码

| Code | 说明 |
|---|---|
| 200 | 操作成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或 Token 已过期 |
| 403 | 没有权限 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 1001 | 用户名已存在 |
| 1002 | 用户不存在 |
| 1003 | 密码错误 |
| 1004 | 账号已禁用 |
| 2001 | 无效的代理 Key |
| 2002 | 代理 Key 已禁用 |
| 2003 | 配额已用完 |
| 3001 | 请求过于频繁 |
| 3002 | 预算已用完 |
| 4001 | 上游服务调用失败 |
| 4002 | 模型不可用 |
| 4003 | 没有可用的 API Key |

## 二、认证模块 `/api/auth`

### 2.1 登录
```
POST /api/auth/login
```

**Request Body**
```json
{
  "username": "admin",    // 必填，用户名
  "password": "123456"    // 必填，密码
}
```

**Response**
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
    "userId": 1,
    "username": "admin",
    "nickname": "Admin",
    "token": "eyJhbGciOi..."
  }
}
```

### 2.2 注册
```
POST /api/auth/register
```

**Request Body**
```json
{
  "username": "zhangsan",     // 必填，3-50位
  "password": "123456",       // 必填，6-20位
  "nickname": "张三",         // 可选
  "email": "zs@example.com", // 可选
  "phone": "13800000000"     // 可选
}
```

### 2.3 登出
```
POST /api/auth/logout
Authorization: Bearer {token}
```

## 三、用户管理 `/api/system/users`

### 3.1 分页查询用户
```
GET /api/system/users?current=1&size=10&keyword=张
```

| 参数 | 类型 | 必填 | 默认 | 说明 |
|---|---|---|---|---|
| current | int | 否 | 1 | 页码 |
| size | int | 否 | 10 | 每页条数 |
| keyword | string | 否 | | 用户名/昵称模糊搜索 |

### 3.2 获取单个用户
```
GET /api/system/users/{id}
```

### 3.3 创建用户
```
POST /api/system/users
```
Body 同注册接口。

### 3.4 更新用户
```
PUT /api/system/users/{id}
```
```json
{
  "nickname": "新昵称",
  "email": "new@example.com",
  "phone": "13900000000",
  "avatar": "https://...",
  "status": 1
}
```

### 3.5 删除用户
```
DELETE /api/system/users/{id}
```

```

### 5.4 删除 Key
```

```

```
PUT /api/apikey/pool/{id}/status

## 八、路由规则 `/api/router/rules`

### 8.1 分页查询
```
GET /api/router/rules?current=1&size=10
```

### 8.2 创建规则
```
POST /api/router/rules
```
```json
{
  "ruleName": "长文本用DeepSeek",
  "conditionExpr": "#promptLength > 2000",     // SpEL 表达式
  "targetProvider": "deepseek",
  "targetModel": "deepseek-chat",
  "priority": 10,
  "fallbackProvider": "openai",
  "fallbackModel": "gpt-3.5-turbo"
}
```
**SpEL 可用变量**：`#model`（请求模型名）、`#promptLength`（prompt 字符长度）

### 8.3 更新规则
```
PUT /api/router/rules/{id}
```

### 8.4 删除规则
```
DELETE /api/router/rules/{id}
```

### 8.5 启停规则
```
PUT /api/router/rules/{id}/status?status=0
```

## 十、用量统计 `/api/stats`

### 10.1 统计概要
```
GET /api/stats/summary?projectId=1&startDate=2026-04-01&endDate=2026-04-13
```

| 参数 | 说明 |
|---|---|
| projectId | 可选，按项目筛选 |
| teamId | 可选，按团队筛选 |
| provider | 可选，按厂商筛选 |
| model | 可选，按模型筛选 |
| startDate | 可选，开始日期 yyyy-MM-dd |
| endDate | 可选，结束日期 yyyy-MM-dd |

**Response**
```json
{
  "data": {
    "totalRequests": 1500,
    "totalPromptTokens": 500000,
    "totalCompletionTokens": 200000,
    "totalTokens": 700000,
    "totalCost": 12.500000,
    "cacheHitCount": 120,
    "cacheHitRate": 8.0
  }
}
```

### 10.2 趋势数据
```
GET /api/stats/trend?startDate=2026-04-06&endDate=2026-04-13
```

### 10.3 按模型统计
```
GET /api/stats/by-model?startDate=2026-04-06&endDate=2026-04-13
```

### 10.4 实时统计
```
GET /api/stats/realtime/{projectId}
```
