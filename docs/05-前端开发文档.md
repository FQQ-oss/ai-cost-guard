| 项 | 值 |
|---|---|
| 框架 | Vue 3.4+ (Composition API `<script setup>`) |
| 构建 | Vite 5.4+ |
| UI 库 | Element Plus 2.7+ |
| 图表 | ECharts 5.5+ |

## 三、启动方式

```bash
cd frontend
npm install       # 首次安装依赖
npm run dev       # 启动开发服务器（端口 3000）
npm run build     # 生产构建
npm run preview   # 预览生产构建
```

## 四、Vite 配置

```js
// vite.config.js
export default defineConfig({
  plugins: [vue()],
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  },
  resolve: {
    alias: { '@': '/src' }
  }
})
```

- 所有 `/api/*` 请求代理到后端 8080 端口
- `@` 别名指向 `src/` 目录

## 五、Axios 封装（request.js）

```
创建 Axios 实例
  ├── baseURL: '/api'
  ├── timeout: 30000ms
  │
  ├── 请求拦截器
  │   └── 从 localStorage 读取 token → 设置 Authorization: Bearer {token}
  │
  └── 响应拦截器
      ├── 成功：解包 response.data
      │   ├── code === 200 → 返回 res
      │   ├── code === 401 → 清除 token → 跳转 /login
      │   └── 其他 → ElMessage.error(res.msg) → reject
      └── 失败：ElMessage.error → reject
```

### 路由守卫

```
beforeEach：
  if (目标不是 /login && 没有 token)
    → 重定向到 /login
  else
    → 放行
```

> 注意：当前没有角色权限守卫，所有已登录用户可访问所有页面。

## 九、页面功能详细

### 9.1 Login.vue（登录页）

- 全屏居中卡片，Element Plus 表单
- 字段：用户名、密码
- 登录成功后存储 token/userInfo，跳转 `/`
- **待完善**：缺少注册入口

### 9.2 Layout.vue（布局）

- 左侧深色侧边栏（#001529），220px 宽
- 9 个菜单项对应 9 个子路由
- 顶栏显示页面标题 + 用户昵称 + 退出按钮
- 内容区灰色背景，`<router-view />` 渲染子页面

### 9.3 Dashboard.vue（数据看板）

- **第一行**：4 个统计卡片（今日请求数、Token 消耗、花费、缓存命中率）
- **第二行左**：请求量趋势（近 7 天）— 柱状图（请求量） + 折线图（花费）双轴
- **第二行右**：模型用量分布 — 环形饼图
- 数据来源：`getStatsSummary`（当天）、`getStatsTrend`（近 7 天）、`getStatsByModel`（近 7 天）

### 9.4 Projects.vue（项目管理）

- 表格列：ID、项目名、团队ID、描述、状态、操作
- 操作：生成代理 Key（弹窗显示 acg-xxx）、删除
- 新建对话框：项目名、团队ID、描述
- 分页

### 9.5 KeyPool.vue（Key 池管理）

- 表格列：ID、厂商、备注、Key（加密后）、RPM 限制、状态（开关）、操作
- 操作：删除、启停切换
- 新建对话框：厂商（下拉：OpenAI/Claude/DeepSeek）、备注、API Key（密码输入框）、Base URL、RPM
- 无分页（一次加载 100 条）

### 9.6 Pricing.vue（模型定价）

- 表格列：厂商、模型、输入价格、输出价格、币种、操作
- 操作：删除
- 新建对话框：厂商、模型、输入价格（6 位小数）、输出价格

### 9.7 RouteRules.vue（路由规则）

- 表格列：ID、规则名、条件表达式、目标厂商、目标模型、优先级、降级模型、操作
- 操作：删除
- 新建对话框（600px 宽）：规则名、条件表达式（placeholder: `#promptLength > 2000`）、目标厂商/模型、优先级、降级厂商/模型

### 9.8 Stats.vue（用量统计）

- **筛选栏**：项目ID、开始日期、结束日期、查询按钮
- **统计卡片**：4 个（同 Dashboard）
- **趋势图**：三系列双轴 — 柱状"请求数" + 柱状"Token消耗" + 折线"花费"
- 默认查近 30 天

### 9.9 Alerts.vue（告警管理）

- **上半部分**：告警规则表格（规则名、类型 Tag、项目ID、阈值、操作删除）+ 添加按钮
- **下半部分**：告警历史表格（规则ID、内容、通知状态 Tag、时间）
- 新建对话框：规则名、类型（下拉：预算/突增）、项目ID、阈值

### 9.10 Audit.vue（审计日志）

- 筛选：操作类型（文本输入）+ 查询
- 表格列：ID、操作人、操作、资源、详情（超长 tooltip）、IP、时间
- 只读，分页

### 9.11 Users.vue（用户管理）

- 表格列：ID、用户名、昵称、邮箱、状态 Tag、创建时间、操作
- 操作：删除
- 新建对话框：用户名、密码、昵称、邮箱
- 分页
